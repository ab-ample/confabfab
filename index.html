Git useful commands

Add a change

  1. git add . This command prepares the changes to do the "commit".
  2. git commit -m "commit title" This command "commits" the changes on the files you have previously "staged".
  3. git push origin This command uploads the changes/commits to the remote repository (Github).

Update local repository

  1. git pull origin This command updates your local repository with the changes from Github repository.
